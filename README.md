# Neoscry

Neoscry — приложение на Python для записи диалога из двух источников и получения точной транскрипции (AI):

- Ваш голос: микрофон
- Голос собеседника: системный аудиовыход (WASAPI loopback на Windows)

Результат можно экспортировать в `txt` (удобно читать) и `json` (структура + таймкоды).

## Возможности

- Два источника: микрофон (вы) + собеседник (выход / loopback / виртуальный input)
- Режимы источника собеседника: input-устройство, loopback-input, или loopback от playback-устройства
- Лайв транскрипция (черновик) + финальная транскрипция после остановки (максимальная точность)
- Экспорт: `sessions/<timestamp>/transcript.txt`, `sessions/<timestamp>/transcript.json`
- Ускорение на GPU: выбор `cuda/cpu/auto` и `compute` (например `float16`)

## Быстрый старт (Windows)

1) Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

2) Запуск:

```bash
python run.py
```

## Как пользоваться

1) Выберите устройства:

- `Записывать микрофон` + устройство микрофона
- `Записывать выход` + источник собеседника (через выпадающий список)

2) Выберите распознавание:

- `Модель`: для максимальной точности — `large-v3`
- `Устройство`: `cuda` (если доступно) или `cpu`
- `Compute`: для CUDA обычно `float16`

3) (Опционально) включите `Лайв транскрипция` — это черновик во время записи.

4) Нажмите `Старт записи`, затем `Стоп + транскрибировать` для финального результата.

## Где лежат результаты

После каждой сессии создается папка `sessions/<YYYYMMDD-HHMMSS>/`:

- `mic.wav` — запись микрофона (если включено)
- `loopback.wav` — запись собеседника (если включено)
- `transcript.json` — сегменты (таймкоды, спикер, текст)
- `transcript.txt` — красиво отформатированный текст

## Примечания

- Для высокой точности рекомендуется модель Whisper `large-v3` (требует больше времени/ресурсов).
- На Windows loopback/виртуальные устройства зависят от драйверов (Voicemeeter и т.п.).
